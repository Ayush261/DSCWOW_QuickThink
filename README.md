<h3>To install and configure Android application development tool on Windows operating systems</h3><br>
<a href="https://developer.android.com/studio"> Download a Android Studio.</a> <br>

### To Download and Installation Pathway Android Studio on Windows, proceed as follows:<br>
             
1.)	Configuration of the computer or Pc must be as follows-<br>
a)	Microsoft Windows 7/8/10 (32-bit or 64-bit)<br>
b)	3 GB RAM minimum, 8 GB RAM recommended (plus 1 GB for the Android Emulator)<br>
c)	2 GB of available disk space minimum, 4 GB recommended (500 MB for IDE plus 1.5 GB for Android SDK and emulator system image).
<br>
d)	1280 x 800 minimum screen resolution<br>

2.)	The Steps to followed for installation of the Android Studio are-<br>
a)	Open the browser of the computer and type the Install the android studio visit to the Google developer website link for the download.<br>
b)	 Website Click on the download android studio button .Click on the “I have read and agree with the above terms and conditions” checkbox followed by the download button. Click on save file button in the appeared prompt box and the file will start downloading.<br>
c)	After the downloading has finished, open the file from downloads and run it.<br>

3.) It will prompt the following dialogue box. Click on next .In the next prompt it’ll ask for a path for installation. Choose a path and hit next.<br>
d)	It will start the installation, and once it is completed. Click on next.<br>
e)	Once “Finish” is clicked, it will ask whether the previous settings needs to be imported [if android studio had been installed earlier], or not. It is better to choose the ‘Don’t import Settings option’. Click the OK button.<br>
f)	This will start the Android Studio. Meanwhile it will be finding the available SDK components.<br>
g)	After it has found the SDK components, it will redirect to the Welcome dialog box .Click on next .Choose Standard and click on Next. Now choose the theme, whether light theme or the Dark one. The light one is called the IntelliJ theme whereas the dark theme is called Dracula. Choose as required. Click on the Next button.<br>
h)	Now it is time to download the SDK components. Click on Finish. It has started downloading the components. The Android Studio has been successfully configured. Now it’s time to launch and build apps.Click on the Finish button to launch it.<br>
i)	Click on ‘Start new android project’ to build a new app.<br>
j)	Start performing a work on the android studio.<br>
